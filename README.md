# Two-Player Hangman Game

A simple two-player Hangman game built with Python and Tkinter.

## How to Play

1. **Player 1** enters a secret word (letters only).  
2. **Player 2** tries to guess the word by entering one letter at a time.  
3. Player 2 has 11 turns to guess the word correctly.  
4. Click "Play Again" to start a new game.

## Requirements

- Python 3.x  
- Tkinter (comes pre-installed with Python)
